# ComfyUI-Avatary

A collection of ComfyUI nodes to speed up your process and reduce headaches.

## Nodes

### Carousel Split Avatary

Instantly breaks carousel and grid outputs into clean native-size panels with a marked preview of every cut.

### Prompt List Avatary

Run multiple prompts from a single list

### Load Images Avatary

Drop, paste, or point at a folder and feed ordered image sets straight into your graph.

### Features Avatary

Adds clean feature flags so experiments can be enabled, disabled, or bypassed fast.

### Save Image Ultra Avatary

Saves final images with cleaner names, prompt metadata, and fewer output-folder chores.

### Aspect Ratio Avatary

Converts presets or custom overrides into exact width and height values.

### Group Bypasser Avatary

Toggles whole workflow sections from one place for faster iteration.

### Nano Banana Pro Avatary

Generates or edits images through Nano Banana Pro directly inside your ComfyUI graph.

### Switch Avatary

Routes variations through one clean switch instead of rewiring your graph.

### ControlLight Avatary

Rescues dark images with ControlLight enhancement while keeping the node simple.

### Aspect Ratio Selector Avatary

Picks reusable preset sizes for any graph section that needs consistent dimensions.
